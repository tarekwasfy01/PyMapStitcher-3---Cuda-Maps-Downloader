# PyMapStitcher 3:

PyMapStitcher 3 is a desktop application for downloading, stitching, and exporting very large satellite map areas as GeoTIFF/BigTIFF files. 
The software supports GPU acceleration with NVIDIA CUDA and CuPy, direct GeoTIFF georeferencing, WebView-based map selection,
and high-performance tile processing for large-scale mapping workflows. 
https://doi.org/10.5281/zenodo.20480746

# Install from PyPI

Install
py -m pip install pymapstitcher
Start
pymapstitcher
or:

py -m pymapstitcher
The package also installs the alias:

pymapstitcher3
Optional CUDA support
CUDA stitching uses CuPy when available. It is optional because CUDA/CuPy wheels depend on the NVIDIA driver/CUDA generation.

For CUDA 12.x, install:

py -m pip install "pymapstitcher[cuda]"
If CuPy is unavailable, the application falls back to CPU stitching.

pymapstitcher

# Features:

Satellite map area selection in an interactive web map
Direct GeoTIFF / BigTIFF export
Embedded georeferencing (EPSG:3857)
NVIDIA CUDA acceleration with CuPy
Multi-threaded tile downloading
Large-area stitching with low memory usage
PySide6 desktop interface
Portable installation with bundled Python environment
