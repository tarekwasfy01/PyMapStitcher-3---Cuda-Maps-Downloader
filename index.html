About

Large-area maps without heavy RAM usage

PyMapStitcher 3 is a desktop application for downloading, stitching, and exporting very large satellite map areas as georeferenced GeoTIFF or BigTIFF files. It is designed for large-area remote sensing and mapping workflows where normal map downloaders often run into memory limits.

Main features

Mapping, CUDA acceleration, and GIS-ready export

  • Interactive satellite map area selection
  • Large-area map tile downloading
  • Low-RAM stitching workflow
  • NVIDIA CUDA / CuPy acceleration support
  • GeoTIFF and BigTIFF export
  • Embedded georeferencing with EPSG:3857 workflows
  • Modern PySide6 desktop interface
  • Portable / offline installer workflow
